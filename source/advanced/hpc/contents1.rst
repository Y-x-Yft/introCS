Performance Engineering of Software Systems (6.172)
----------------------------------------------------------
This course provides a hands-on, project-based introduction to building **scalable** and **high-performance** software systems.

| Topics include performance analysis, algorithmic techniques for high performance, instruction-level optimizations, caching optimizations, parallel programming, and building scalable systems.

* programming language: C

* `homepage <https://ocw.mit.edu/courses/electrical-engineering-and-computer-science/6-172-performance-engineering-of-software-systems-fall-2018/syllabus/>`_
